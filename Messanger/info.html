Создание социальной сети

1.Введение
Сегодня Интернет проник во все сферы нашей жизни. Он заменил нам книги и телевизор. Благодаря нему мы имеем неограниченный доступ к информации. Но в основном его используют как средство связи. В этой статье я расскажу про создание социальной сети.
Эта социальная сеть создавалась для моего портфолио для работы, но была адаптирована для проекта. Её изначально планировалось сделать просто как простой текстовый чат, однако он стал постепенно модернизироваться и в конце-концов стал социальной сетью, которой даже можно пользоваться. Данная социальная сеть не является коммерческим проектом.
2.Постановка проблемы
Само по себе создание сайта - задача не сложная. Можно воспользоваться конструктором сайтов или немного изучить некоторые технологии. Для создания более сложных сайтов конструктор уже не подойдёт. А уже для создания небольных социальных сетей (как у меня) нужно хорошо разбираться в тонкостях работы.
3.Цель проекта
Ознакомить с алгоритмом и технолониями, использованными при создании проекта
4.Задачи проекта
  • Ознакомить с алгоритмом создания соц. сети
  • Ознакомить с используемыми технологиями
  • Рассказать о возможных трудностях
  • Опубликовать сайт в Интернете
5.Ожидаемый результат
Простая рабочая социальная сеть или любой другой веб-сервис
6.Этапы работы
Часть 1. Планирование
Для начала нужно определится для с какой целью создавать социальную сеть. Например можно её сделать для простых переписок, можно для знакомств,а можно для чего-то большего... Для чего была создана это социальная сеть было сказано выше
Когда определились с вектором развития необходимо определится с технологиями, которые будем использовать при разработке.
Часть 2. Технологии (теория)
При разработке данной соц сети были использованы языки HTML,CSS, JavaScript. Пойдём по порядку
HTML - язык гипертекстовой разметки. Это базовый язык для создания веб-страниц. Благодаря нему создаются все элементы сайта: заголовки, картинки, колонки с текстом, поля ввода и др. Говоря простым языком, это "скелет" сайта. Учится очень легко, если захотеть можно выучить за пару часов. Теоретически, можно создать сайт с только с помощью этого языка, но сайт получится статичным и без какого-либо функционала.
CSS - язык описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки (выше упомянутого HTML). С его помощью сайт становится красивым. Нажмите чтобы увидеть этот сайт без CSS. Это наглядный пример для чего нужен CSS
JavaScript - язык программирования. С него и начинается самое сложное. Он нужен для создания логики сайта. Например, кнопка выше работает благодаря этому языку. Изучение может быть очень затруднительным. Также на этом языке написана логика серверная составляющая этого сайта, с использованием технологии Node JS с использованием множества npm-пакетов . Клиент написан на Vue JS. Понимание того, что я написал выше, придёт с опытом.
MySQL - база данных. Ничего особенного в ней нет.
Часть 3. Практика
Для начала нужно изучить HTML и CSS. HTML можно выучить за час, а базовый CSS - за месяц. В источниках будут указаны ресурсы, которыми я пользовался при изучении. Поттом необходима практика. Можно просто создать пару небольших сайтом, чтобы закрепить полученные умения.
Далее нужно найти или создать дизайн для сайта, и научиться пользоваться линейкой и сеткой в Photoshop. Опять же, все умения придут с опытом. Я не буду рассказывать все подробности, так как презентация получится слишком большой.
Самая сложная часть - написание логики. Чтобы избежать проблем, связынных с ней, нужно тщательно продумать все детали: какие кнопки за что должны отвечать, куда какие и данные отправлять и т п.
