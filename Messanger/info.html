Создание социальной сети

Эта часть индивидуального проекта посвящена созданию социальной сети.В этой статье я буду использовать сложные термины, но постараюсь объяснить чтобы было понятно.
Часть 1. Планирование
Для начала нужно определится для с какой целью создавать социальную сеть. Например можно её сделать для простых переписок, можно для знакомств,а можно для чего-то большего... Концретно эта социальная сеть создавалась для моего портфолио для работы, но была адаптирована для проекта. Её изначально планировалось сделать просто как простой текстовый чат, однако он стал постепенно модернизироваться и в конце-концов стал социальной сетью на подобии Viber (Viber это приложение-мессенджер, при создании функционала я ориентировался на него).
Когда определились с вектором развития необходимо определится с технологиями, которые будем использовать при разработке.
Часть 2. Технологии
При разработке данной соц сети были использованы языки HTML,CSS, Javascript. Пойдём по порядку
HTML - язык гипертекстовой разметки. Это базовый язык для создания веб-страниц. Благодаря нему создаются все элементы сайта: заголовки, картинки, колонки с текстом, поля ввода и др. Говоря простым языком, это "скелет" сайта. Учится очень легко, если захотеть можно выучить за пару часов. Теоретически, можно создать сайт с только с помощью этого языка, но сайт получится статичным и без какого-либо функционала.
CSS - язык описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки (выше упомянутого HTML). С его помощью сайт становится красивым. Нажмите чтобы увидеть этот сайт без CSS. Это наглядный пример для чего нужен CSS
